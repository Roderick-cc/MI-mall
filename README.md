# 移动端商城

本项目是基于vue+mint-UI+mui+axios+webpack-CLI 开发的, 主要还是运用vue全家桶的大部分技术。相对于商城来说一般会涉及大量的数据交互，所以这里采用的是json来模拟实现数据交互。

 建议在F12移动端调试模式下或在手机浏览器下观看

模拟测试账号：vue    密码：123

### 安装依赖包

```
npm install
```

### 开启实时编译

```
//运行项目，在本地浏览器访问http://localhost:8080
npm run dev
```

### 功能模块

+ 六大模块（首页，商品分类，商品推荐，购物车，用户界面，登录界面）
+ 模拟购物车商品增加删除、结算金额、下单
+ 用户数据状态保持
